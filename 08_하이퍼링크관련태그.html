[용어 정리]
    Hyper : 초월
    Link : 연결
    Hyper Link : 현재 문서를 넘어 다른 문서와 연결
    Hyper Text : 클릭시 연결된 다른 문서로 넘어가는 글자(문자열)

    HTML (Hyper Text Markup Language)
    : 다른 문서와 연결될 수 있는 글자를 작성할 수 있고
      태그 형태를 해석해서 화면에 출력하는 언어

    HTTP (Hyper Text Transfer Protocol)
    : 인터넷을 Hyper Text가 작성된 문서(html)를
      서로 전송할 때의 약속

a(Anchor, 연결)

    - 현재 문서(html)와 다른 문서(html)를 연결 시켜
      클릭 시 이동하게 만드는 태그

    - a태그로 감싸진 글/내용/이미지 == Hyper Text

    - Hyper Text는 다른문서와 구분되는 html만의 핵심기능(이였던것 다들 잘쓰는중)

a 사용법1 : 페이지 이동

    - href(Hyper Text REFerence) : 연결할 주소를 작성하는 속성
    - target : a 클릭시 현재탭, 새탭 중 어떻게 열것인지 지정
  1. 글자관련으로 이동..
  2. 목록관련으로 이동..
  3. 연습문제1으로 이동..

페이지 내에서 점프하기

    - 한페이지 내에서 지정된 id 속성 값을 가진 요소로 이동
    * id(Identifier, 식별자) 속성
      - 요소를 구분하는 용도의 속성
      - 하나의 html 파일 내에
        같은 id 값을 지닌 요소가 있어서는 안된다
